About

System information and details

System Information

Application Air Quality Monitor
Version 2.0.0 (Lightweight)
Platform Raspberry Pi 3
UI Framework Alpine.js + Tailwind CSS
API Server Flask REST API

Features

  • Real-time air quality monitoring
  • PM2.5 and PM10 particle monitoring
  • AQI v2 calculations (atmospheric)
  • Historical data tracking
  • Automated scheduling
  • Mobile responsive design
  • Dark mode support
  • Lightweight for RPi3 performance

API Endpoints
